# 🐍 Multi-player Snake Game & Chat Room (Victor Game)

歡迎來到由 **Victor Game** 開發的多人遊戲平台。

### 🚀 快速上手
1.  進入此 Space 即可開始遊戲。
2.  輸入您的暱稱，即可在畫面中看到您的蛇。
3.  使用方向鍵或 WASD 操作移動。
4.  上方的對話框可以與同一伺服器的玩家即時聊天。

### 🛠️ 技術棧
*   **後端**: Python Flask, Flask-SocketIO, Eventlet
*   **前端**: HTML5 Canvas, CSS (Glassmorphism), JavaScript
*   **部署**: Hugging Face Spaces (Docker)
